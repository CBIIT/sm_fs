The range of priority scores for applications being recommended for funding under this funding plan. The fundable range begins with the best scored application in this RFA/PA and should end with the priority score that is the "natural" cut-off for the funding selections. Applications selected outside the recommended fundable range will be marked as "Exceptions". Applications within the recommended score range not selected for funding will be marked as "Skips". Justifications for Exceptions and Skips must be provided in the funding plan. Please note that the funding plan will be rejected if you have not provided justification for each grant application marked as "Exception" or "Skip".
